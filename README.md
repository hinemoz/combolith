# Combolith

格闘ゲームのコンボエディタプロジェクト。

「Combo（コンボ）」+「Monolith（一枚岩・礎）」を組み合わせた名前で、
コンボ構築の土台となるツールを目指します。

## 概要
- コントローラー風UI（レバー9方向＋ボタン8個）でコンボを直感的に入力
- 入力すると自動で「＞」区切りが挿入され、テキストボックスに追記される
- テキストボックスは直接編集も可能
- コピー ボタンでクリップボードへコピー
- システム入力（ディレイ・キャンセル・ラッシュ等の任意の文字列）をワンタップ入力、自由に追加・削除可能
- 必殺技ライブラリ（ゲーム→キャラクター→技の階層でコマンドを登録し、ワンタップでコンボに挿入）
- ボタンのラベル・表記はユーザーが自由にカスタマイズ可能
- アプリ内ヘルプ（使い方・データ保存についての説明）を搭載
- ログイン機能なし。データはブラウザの localStorage にのみ保存

## 技術スタック
- React + TypeScript + Vite
- Tailwind CSS
- localStorage（バックエンドなし、Web公開のみを想定した軽量構成）

## データ保存について
このアプリは**ブラウザの localStorage** にすべてのデータを保存しています。サーバーやアカウント機能は一切なく、以下の特徴があります。

- 保存場所: 今使っているブラウザ・端末の中だけ（他の端末やブラウザとは共有されません）
- 保存タイミング: コンボテキストやボタン設定、必殺技ライブラリ等を変更するたびに自動保存
- 消えるケース: ブラウザの「閲覧データを削除（サイトデータ／Cookie削除）」を行うと、保存内容も消えます。シークレット/プライベートウィンドウでは閉じると消えます
- バックアップ・引き継ぎ方法: 現状は自動エクスポート機能がないため、他端末に移したい場合はコンボテキストのコピー＆貼り付け等、手動での移行が必要です

保存に使われている localStorage のキー一覧（開発者向け）:
| キー | 内容 |
| --- | --- |
| `combolith.comboText.v1` | コンボテキスト本文 |
| `combolith.buttons.v1` | ボタンのラベル・表記設定 |
| `combolith.favorites.v1` | システム入力の登録内容 |
| `combolith.moveLibrary.v1` | 必殺技ライブラリ（ゲーム・キャラ・技） |

## ディレクトリ構成
```
combolith/
├── 仕様書.md        # 元の仕様メモ
├── README.md
├── app_backup/      # 必殺技ライブラリ機能追加前のバックアップ
└── app/             # アプリ本体 (Vite プロジェクト)
    └── src/
        ├── App.tsx            # メイン画面
        ├── types.ts           # 型定義・デフォルト値
        ├── storage.ts         # localStorage 読み書き
        └── components/
            ├── DirectionPad.tsx    # レバー入力(9方向)
            ├── ButtonPad.tsx       # ボタン入力(8ボタン)
            ├── FavoritesBar.tsx    # システム入力バー
            ├── ButtonSettings.tsx  # ボタン名称設定モーダル
            ├── MoveLibrary.tsx     # 必殺技ライブラリ
            └── HelpModal.tsx       # アプリ内ヘルプ（使い方・データ保存説明）
```

## 開発の始め方
```bash
cd app
npm install
npm run dev      # 開発サーバー起動 (http://localhost:5173)
npm run build    # 本番ビルド (dist/ に出力)
```
